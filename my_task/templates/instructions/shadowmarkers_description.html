Shadow Marks

A Shadow Mark is an object that can be placed in one display, and will be shadowed (or mirrored) in all other displays at the same location. An example Shadow Mark is the red horizontal line visible in all of the videos of Figure 1.

Fig.1. Example of a Shadow Mark placed within a plant video and shadowed in all other plant videos.


In this example, the user has placed a Cursor Shadow Mark at the top of the first plant by clicking and dragging within the display. The system duplicates the line at the same location in all the other videos. Now the user can compare the heights of all the plants by observing which plants are above or below the line.
Several types of Shadow Marks are available to help with different types of tasks. These modes are available to the right of the scrollbar, as seen in Figure 2.

Cursor: Shadow Mark that follows your cursor as you move around a display.


Freeform: Shadow Mark that allows you to freely draw within a display.


Line: Shadow Mark that allows you to draw a line within a display.


Circle: Shadow Mark that allows you to draw a circle within a display.


Rectangle: Shadow Mark that allows you to draw a rectangle within a display.


Marker: Shadow Mark that resembles a + symbol and can be placed anywhere within a display.
Fig.2. Each Shadow Mark mode.
