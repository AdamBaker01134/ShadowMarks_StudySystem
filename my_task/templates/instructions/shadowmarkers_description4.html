Shadow Marks

A Shadow Mark is an object that can be placed in one display, and will be shadowed (or mirrored) in all other displays at the same location. An example Shadow Mark is the red cursor visible in all of the scatterplots of Figure 1.

Fig.1. Example of a Shadow Mark placed within a scatterplot and shadowed in all other plots.


In this example, the user positions a Cursor Shadow Mark at the highest outlier in the second scatterplot by hovering within the display. The system duplicates the cursor lines at the same location in all the other images. Now the user can compare the heights of all the data by observing which points are above or below the line.

You zoom in by holding CONTROL and pressing the + key, and zoom out by holding CONTROL and pressing the - key.

If you forget any of the features available, click on the help button located in the bottom left corner.
